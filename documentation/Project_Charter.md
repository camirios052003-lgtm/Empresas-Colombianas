# Project Charter - Análisis Financiero de las 10.000 Empresas Más Grandes de Colombia

**Nombre del proyecto:**

**Autor**

Laura Camila Rios

**Fecha**

Julio 2026

---

## 2. Descripción

Este proyecto desarrolla un análisis financiero de las 10.000 empresas más grandes de Colombia utilizando información publicada por la Superintendencia de Sociedades.

El proyecto integra procesos de ingeniería y análisis de datos para transformar información financiera en indicadores y visualizaciones que apoyen la toma de decisiones.

---

## 3. Objetivo general

Analizar el desempeño financiero de las empresas colombianas mediante técnicas de análisis de datos para identificar patrones, tendencias y oportunidades de negocio.

---

## 4. Objetivos específicos

- Comprender la estructura de los datos.
- Desarrollar un proceso ETL.
- Construir una base de datos en SQL Server.
- Calcular indicadores financieros.
- Desarrollar un dashboard en herramienta de visualización.
- Documentar completamente el proyecto en GitHub.

---

## 5. Fuente de datos

Superintendencia de Sociedades

Informe de las 10.000 empresas más grandes de Colombia.

Corte 2025.

## 7. Alcance

El proyecto analizará:

- Ingresos
- Utilidades
- Activos
- Pasivos
- Patrimonio
- Sectores económicos
- Regiones
- Ranking empresarial

No se realizarán modificaciones sobre la base original.

---

## 8. Alcance funcional

El proyecto comprenderá las siguientes etapas:

1. Comprensión del negocio.
2. Comprensión y perfilamiento de los datos.
3. Limpieza y transformación de datos (ETL).
4. Modelado y almacenamiento en SQL Server.
5. Análisis exploratorio de datos (EDA).
6. Construcción de indicadores financieros.
7. Desarrollo de un dashboard interactivo en Power BI.
8. Documentación técnica y publicación en GitHub.

## 9. Indicadores a desarrollar

Durante el proyecto se calcularán indicadores financieros como:

- Crecimiento de ingresos (%)
- Crecimiento de utilidades (%)
- Margen neto
- ROA (Retorno sobre Activos)
- ROE (Retorno sobre Patrimonio)
- Razón de endeudamiento
- Participación por sector
- Participación por región
- Ranking por rentabilidad
